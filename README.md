# Math
## General
◯ [Math for Game Devs, YT series by Freya Holmer](https://youtube.com/watch?v=fjOdtSu4Lm4&list=PLImQaTpSAdsArRFFj8bIfqMk2X7Vlf3XF&index=0): great explainer but lots of down-time in videos you'll want to skip
